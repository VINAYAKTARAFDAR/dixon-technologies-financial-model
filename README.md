# Dixon Technologies Financial Model & DCF Valuation

## Overview
Built an integrated 3-statement financial model and DCF valuation model for Dixon Technologies using historical financial statements and forecasting methodologies.

## Tools Used
- Microsoft Excel
- Financial Modeling
- DCF Valuation
- FCFF Analysis
- Forecasting

## Key Features
- Revenue and margin forecasting
- Integrated Income Statement, Balance Sheet, and Cash Flow Statement
- FCFF valuation model
- WACC calculation
- Scenario and sensitivity analysis

## Project Highlights
- Structured assumptions-driven forecasting model
- Analyzed operating performance and valuation drivers
- Developed valuation framework for investment analysis
